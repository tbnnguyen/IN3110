# Assignment 6

### data.py
The script is not intended to run on its own, but the modules to be used in other scripts. 

To run the program:
```
python3 data.py diabetes.csv
```

### fitting.py
Script applies the models to the data and saves the models as an image. 

It can be called from the terminal as 
```
python3 fitting.py diabetes.csv
```

### models.py
The module performs instantiating and fitting of data on different models.

### visualize.py
Uses Matplotlib to to visualize graphs.
